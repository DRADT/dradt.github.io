Edit Text

Edit text is a notepad text editor since it allows us to edit texts in txt, py, java, cc format and in addition to a .edit format, that format allows you to save styles and also save events such as when you exit and edit mode to edit and not show things when opening and url, it also allows you to insert images, when opening each file it will allow you to see its description and its author.
